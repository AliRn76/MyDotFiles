# MyDotFiles
my configs for nvim and ...


for adding nvim config to your system just copy nvim to your ```.config``` folder and install plugins with [vim-plug](https://github.com/junegunn/vim-plug)
and [packer](https://github.com/wbthomason/packer.nvim) for installing lua packages and have fun 
install lang-servers with [coc](https://github.com/neoclide/coc.nvim)
